# MAAN
Code for the paper "Marginalized Average Attentional Network for Weakly-Supervised Learning" (ICLR).

[[PDF]](https://openreview.net/pdf?id=HkljioCcFQ)

Code is tested with Pytorch 1.0 + Python 3.6.
